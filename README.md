# 🍽️ Recipe Finder

Recipe Finder is a modern full-stack web application that helps users discover, save, and organize recipes through an interactive and personalized platform. The application provides text and voice-based search, recipe categorization, and user-specific features for a seamless cooking experience.

## 🚀 Features

- 🔍 Search recipes by name or ingredients
- 🎙️ Voice-based recipe search
- 🎲 Surprise Me feature for random recipe suggestions
- ❤️ Like and save favorite recipes
- 📤 Share recipes with others
- 📺 Watch recipe videos through YouTube links
- 🥗 Recipe categories:
  - Vegetarian
  - Non-Vegetarian
  - Desserts
  - Drinks
- 👤 User authentication and profile management
- 🌙 Light and Dark theme support
- 🕒 Recently viewed recipes
- 📌 Personalized saved recipes collection

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Recipe-Finder.git
```

### Navigate to the project folder

```bash
cd Recipe-Finder
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
